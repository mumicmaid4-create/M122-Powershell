# M122-Powershell
Ein Script um neue Mitarbeiter automatisch in die AD aufzunehmen

IT-Mitarbeitende können mit dem bereitgestellten PowerShell-Skript neue Benutzerkonten im Active Directory erstellen. Die Erstellung erfolgt über eine grafische Eingabemaske (GUI), welche alle benötigten Informationen abfragt und die Benutzer automatisiert in der korrekten OU anlegt.
